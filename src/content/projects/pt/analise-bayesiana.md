# Análise Bayesiana de Dados Climáticos

Este projeto aplica métodos de inferência bayesiana para analisar séries temporais de dados climáticos.

## Metodologia

Utilizamos o framework Stan para implementar modelos hierárquicos bayesianos.

## Resultados

O modelo demonstrou superior desempenho preditivo em comparação com abordagens frequentistas clássicas.
