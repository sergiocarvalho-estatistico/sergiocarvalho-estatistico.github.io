# Dashboard Interativo de Dados Socioeconômicos

Desenvolvimento de um dashboard interativo que permite explorar indicadores socioeconômicos de municípios brasileiros.

## Funcionalidades

- Mapas coropléticos interativos
- Análise comparativa entre municípios

## Tecnologias

R, Shiny, ggplot2, leaflet, DT
