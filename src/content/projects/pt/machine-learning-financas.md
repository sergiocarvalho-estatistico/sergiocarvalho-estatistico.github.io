# Machine Learning em Finanças

Projeto de desenvolvimento de modelos preditivos para análise de risco de crédito.

## Técnicas Utilizadas

- XGBoost e LightGBM para classificação de risco
- SHAP values para interpretabilidade

## Impacto

Redução de 23% na taxa de inadimplência não detectada em carteira piloto.
